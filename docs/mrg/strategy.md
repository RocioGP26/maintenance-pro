# MRG Strategy · Sprint 10

## Objetivo

**Maintix Reference Guide v1.0 (MRG)** — la documentación funcional oficial del producto.

MAG explica **cómo integrar** la API. MRG explica **cómo funciona** Maintix para quien opera, implementa o consulta el negocio.

## Principios MRG

1. **Lenguaje funcional** — Mantenimiento · Inventario (no nombres técnicos en prosa)
2. **Honestidad de estado** — producción vs roadmap
3. **Sin duplicar MPA/MAG/MSD** — enlazar
4. **Base para capacitación** — manuales de usuario · soporte · comercial
5. **Tenant-first** — toda operación en contexto de empresa
