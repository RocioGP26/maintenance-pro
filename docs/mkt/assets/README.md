## Cómo descargar el PDF

1. Abre el HTML en el navegador (vía `python run.py` → URL de la tabla).
2. Pulsa **Descargar / Imprimir PDF**.
3. En el diálogo de impresión, elige **Guardar como PDF** · tamaño **A4** · márgenes **Ninguno** · activar **Gráficos de fondo**.
