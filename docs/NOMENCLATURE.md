# Nomenclatura · Maintix Documentation Suite

Códigos oficiales del ecosistema documental. **Patrón:** siglas **M** + nombre corto → nombre completo en inglés/español.

| Código | Nombre completo | Carpeta | URL |
|--------|-----------------|---------|-----|
| **MBB** | Maintix Brand Book | `docs/brandbook/` | `/brandbook/` |
| **MDL** | Maintix Design Language | `docs/mdl/` | `/mdl/` |
| **MUX** | Maintix User Experience Guide | `docs/mux/` | `/mux/` |
| **MRL** | Maintix Report Language | `docs/mrl/` | — |
| **MCM** | Maintix Commercial Manual | `docs/mcm/` | Sprint 5 |
| **MAG** | Maintix API Guide | `docs/mag/` | Planificado |
| **MADR** | Maintix Architecture Decision Records | `docs/madr/` | Planificado |

Proyectos sin sigla M (Handbook, Deployment, Security, Roadmap) usan nombre descriptivo en `docs/`.
