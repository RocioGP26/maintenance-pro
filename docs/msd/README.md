## Objetivo

Construir el ecosistema completo para desarrolladores externos: Developer Portal, OpenAPI 3.1, SDK oficiales, CLI, sandbox, guías y colecciones.

**MAG (07)** define qué puede hacer la API. **MSD (08)** entrega cómo integrarla.
